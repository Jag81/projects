Administrator
  • All requests
  • Manager list
Description:
  • My Requests: list a user's (current login) all requests. In the request form the user fills request information and specify a manager for approving
  • My Approvals: list all requests for a manager approval (current login)
  • All requests: list all request in the system for view only or delete
  • Manager list: a manager list for requester to select for approval
  • A user can save a request as a draft or complete and sending alert email to a manager
  • Once the request is approved, the requester cannot edit the request again
